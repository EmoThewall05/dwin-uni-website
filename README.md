# Dwin Universe

Dwin Universe is a high-performance web application featuring advanced AI-powered interactions and a streamlined cryptographic asset swap engine.

## 🚀 Key Features

*   **Emo Key Chat Engine**: An AI-integrated chat interface protected by the Emo Key cryptographic safety layer, ensuring secure communication with the Dwin AI backend.
*   **Wallet Swap Engine**: A sophisticated asset swapping interface built with Framer Motion, featuring smooth, responsive animations and success feedback indicators.
*   **Modern Tech Stack**: Engineered with React, Vite, Tailwind CSS, and Framer Motion for a seamless, professional user experience.

## 🛠 Tech Stack

- **Framework**: React 18+ (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Backend**: Cloudflare Workers (Dwin AI Integration)

## ⚙️ Setup & Development

### Installation

```bash
npm install
```

### Running Locally

```bash
npm run dev
```

### Environment Variables

Ensure the following variables are configured in your environment:

| Variable | Description |
| :--- | :--- |
| `X-Emo-Key` | The active safety key: `emo_guardian_AI_safety_active_2026` |

## 🔒 Security

This application utilizes a proprietary `X-Emo-Key` authentication header (`emo_guardian_AI_safety_active_2026`) to interface with the protected Dwin Cloudflare Worker AI endpoint. Access is restricted and requires valid cryptographic headers.
